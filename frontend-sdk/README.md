Frontend SDK
============

Generate a strict TypeScript client from the API OpenAPI schema.

Prerequisites
- Node.js >= 18

Install
```
cd frontend-sdk
npm i
```

Generate (local API)
```
npm run generate
```

Generate (custom API base)
```
API_SPEC=https://api.example.com/openapi.json npm run generate
```
